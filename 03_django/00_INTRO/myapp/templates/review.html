장고 실행 방법

  1. 만들 곳의 파일로 들어간다
  2. 터미널에 django-admin startproject 프로젝트명 을 실행한다
  3. python manage.py runserver를 실행시켜 서버가 실행되는지 확인한다
  4. python manage.py startapp 어플명 를 실행시켜 application을 만들어 준다.
  5. app을 만들었으면 main project의 settings.py로 들어가서 INSTALLED_APPS 에 app을 추가해 준다

장고에 url 추가

  1. main의 urls에 다음 과 같이 변경

from django.urls import path, include

  1. app에 urls.py파일 생성 후에 다음과 같이 변경

from django.urls import path
# 현재 파일 위치와 같은 위치에
#  views.py 를 import
from . import views
# 그냥 메인파일에서 바로 할려면
from app파일명 import views
#이렇게 하면됨

# myapp/hello
urlpatterns = [
    # myapp/hello/
    path('hello/', views.hello),
    # myapp/hye/ 
    # bye를 입력하면 views에 있는 bye함수를 실행한다.
    path('bye/', views.bye),
]

  1. app의 views에 다음과 같이 변경

from django.shortcuts import render

from django.http import HttpResponse

# view => function

def bye(request):
    return HttpResponse('Goodbye My Friend')

def hello(request):
    return HttpResponse('Hello My Friend')

장고에서 html 사용하기

  1. 사용할 app에 templates 파일 만들어서 html 파일 넣기
  2. app의 views 파일의 함수를 render를 이용하여 다음과 같이 고쳐준다

def nice(request):
    # render의 첫번째 인자 => request 고정
